# Breast-Cancer-Prediction

##  Overview

This project focuses on predicting whether a tumor is **malignant (cancerous)** or **benign (non-cancerous)** using machine learning algorithms. Early detection plays a critical role in improving survival rates, making this a highly impactful healthcare application.

##  Objective

To build and evaluate a classification model that accurately predicts breast cancer diagnosis based on tumor characteristics.

##  Dataset

* Source: Kaggle Breast Cancer Dataset
* Features include radius, texture, perimeter, area, smoothness, etc.
* Target Variable:

  * 0 → Benign
  * 1 → Malignant

##  Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

##  Workflow

1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Scaling
4. Model Training
5. Model Evaluation

##  Models Used

* Logistic Regression

## 📈 Results

* Accuracy: 98%
* Key Insight: Certain features like radius and texture strongly influence tumor classification.

## 📸 Visualizations

* Target Distribution
* Confusion Matrix
